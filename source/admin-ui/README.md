# IM Admin UI

独立后台管理前端，默认连接 `http://127.0.0.1:8000`。

## Start

```powershell
cd D:\desktop\work\admin-ui
npm install
npm run dev
```

打开：

- `http://127.0.0.1:5174`

## Features

- Dashboard overview
- User management and enable/disable
- Online user monitoring
- Message audit
- Friend request audit
