# ALIGNMENT_IMPLEMENT_IM_UI

## 1. 项目上下文
- **目标**: 实现企业级即时通讯应用的前端界面。
- **输入**: `im-ui.txt` (包含15个功能模块的详细需求)。
- **技术栈**: React + TypeScript + Vite + Tailwind CSS + shadcn/ui + Lucide Icons + React Router DOM。
- **当前状态**: 项目脚手架已搭建，shadcn/ui 组件库已修复。

## 3. 架构设计 (简述)
- **路由**: 使用 `react-router-dom` v7。
  - `/auth/*`: 认证相关 (Login, Startup)
  - `/`: 主布局 (包含底部导航)
    - `index`: 会话列表
    - `contacts`: 通讯录
    - `workspace`: 工作台
    - `me`: 我
  - `/chat/:id`: 聊天页面 (单聊/群聊)
  - `/chat/settings/:id`: 聊天设置
  - `/search`: 全局搜索
  - `/files`: 文件管理
  - `/profile/:id`: 用户资料
  - `/settings`: 通用设置
- **状态管理**: 计划使用 `zustand` 管理全局状态 (Auth, Theme, Chat Store)。
- **样式**: Tailwind CSS + CSS Variables (支持深色模式)。
- **图标**: Lucide React。
