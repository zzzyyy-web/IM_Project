# 任务拆分：功能清单实现

## 阶段 1：基础消息功能增强 (High Priority)
1.  **TASK-001: 语音消息实现**
    -   输入区域增加语音按钮，长按录音 UI（波形动画）。
    -   消息气泡支持语音播放（点击播放/暂停，进度条）。
    -   模拟录音时长限制（60s）。
2.  **TASK-002: 文件消息实现**
    -   输入区域增加“文件”入口，支持文件选择。
    -   消息气泡显示文件图标、名称、大小。
    -   模拟文件上传/下载进度条。
3.  **TASK-003: 名片消息实现**
    -   输入区域增加“名片”入口，弹出联系人选择器。
    -   消息气泡显示名片样式（头像、昵称、个人名片）。
    -   点击名片跳转到用户资料页。
4.  **TASK-004: 引用回复与多选转发**
    -   长按消息菜单增加“引用”选项，输入框显示引用内容。
    -   长按消息菜单增加“多选”选项，底部出现转发/删除工具栏。
    -   实现转发弹窗（选择联系人/群组），支持逐条转发/合并转发（Mock 合并文本）。
5.  **TASK-005: Emoji 与贴图**
    -   集成 `emoji-picker-react` 或类似库。
    -   输入框支持插入 Emoji。
    -   发送贴图（图片消息的一种特殊展示）。
6.  **TASK-006: 正在输入与已读状态**
    -   Store 中增加 `typingUsers` 状态。
    -   输入框 `onChange` 触发模拟 socket 事件。
    -   顶部标题栏显示“对方正在输入...”。
    -   消息气泡右下角显示“已读/未读”状态（仅单聊）。

## 阶段 2：群组与联系人管理 (Medium Priority)
7.  **TASK-007: 群组设置完善**
    -   群设置页增加“群名称”、“群公告”编辑入口（仅群主/管理员）。
    -   成员列表支持“添加”（邀请好友）和“移除”（踢人）。
    -   群内昵称设置。
8.  **TASK-008: 联系人添加流程**
    -   全局搜索框支持搜索未添加用户（Mock 数据库）。
    -   用户资料页显示“添加到通讯录”按钮。
    -   “新的朋友”页面显示申请列表（接受/拒绝）。
9.  **TASK-009: 全局搜索**
    -   实现全局搜索组件，支持搜索联系人、群组、聊天记录。
    -   高亮显示搜索结果。

## 阶段 3：系统设置与细节 (Low Priority)
10. **TASK-010: 通用设置**
    -   实现深色模式切换（Tailwind dark mode）。
    -   字体大小调整（CSS 变量）。
    -   多语言切换（i18n 框架搭建）。
    -   通知开关（LocalStorage 持久化）。
11. **TASK-011: 多端登录与网络状态**
    -   模拟网络断开/连接状态提示。
    -   顶部显示“Windows 微信已登录”等状态栏。

## 依赖关系
- TASK-001, TASK-002, TASK-003 依赖基础 Message 组件重构。
- TASK-004 依赖 Store 的多选状态管理。
- TASK-007 依赖 Mock 数据结构的完善。
